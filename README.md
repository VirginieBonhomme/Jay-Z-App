### MVP/PostMVP

#### MVP

- Home page with clickable icons that route to each respective feed
- Get and post lyrics from Airtable.
- Use forms to create lyric and update Airtable.
 

#### PostMVP

- Like your favorite lyric
- Edit post
- Add animations
